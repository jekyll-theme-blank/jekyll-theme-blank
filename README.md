# [Blank](https://jekyll-theme-blank.github.io/)

This is a blank Jekyll theme for GitHub Pages.

## Usage

In your `_config.yml`:

```yaml
remote_theme: jekyll-theme-blank/jekyll-theme-blank
```

See the theme's [_config.yml](_config.yml) for options.
